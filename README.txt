== Description ==
QR Payments For Woocommerce Payment Gateway for the following e-wallets
- Touch N Go
- DuitNow
- Grab
- Shopee Pay
- Boost

This plugin is compatible with WooCommerce HPOS 

Please do take note that this plugin is a manual upload, no api to payment gateway, you are required to use your own QR code.

== Installation ==

1. Be sure you're running WooCommerce 8.2.1 in your shop.
2. You can:(1) upload the entire 'qr-pay-gateway' folder to the '/wp-content/plugins/' directory, 
	   (2) upload the .zip file with the plugin Uploader under > Plugins > Add New > Upload
3. Activate the plugin through the PLUGINS menu in WordPress
4. Go to WooCommerce > Settings > Payment and select "Qr Pay Gateway" to configure
5. Visit our page for more details https://www.equilibrium.my/qr-pay-gateway/
